# Percobaan Rotary Encoder dengan ESP32-S3

## Membaca posisi dan arah putaran dari rotary encoder.

## Komponen
- 1x ESP32-S3
- 1x Rotary Encoder

## Wiring → docs/images/wiring_encoder

## Program → src/encoder/encoder.ino

## Langkah Perrcobaan 
1. Buat New Project
2. Sesuaikan wiring 
3. Upload kode ke ESP32-S3 / jalankan di wokwi
4. Putar knob encoder.
5. Lihat perubahan nilai di Serial Monitor (arah dan nilai counter).
