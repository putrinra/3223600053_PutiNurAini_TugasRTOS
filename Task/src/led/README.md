# Percobaan LED dengan ESP32-S3

## Menguji penggunaan multitasking pada ESP32-S3 untuk mengontrol beberapa LED secara bersamaan, serta mempelajari konsep pembagian tugas antar core

## Komponen
- 1x ESP32-S3
- 3x LED

## Wiring → docs/images/wiring_led

## Program → src/led/led.ino

## Langkah Percobaan 
1. Buat New Project
2. Sesuaikan wiring 
3. Upload kode ke ESP32-S3 / jalankan di wokwi
4. Buka serial monitor
