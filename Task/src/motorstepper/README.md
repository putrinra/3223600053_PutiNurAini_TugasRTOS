# Percobaan Motor Stepper dengan ESP32-S3

## Mengendalikan motor stepper menggunakan ESP32-S3 dan driver

## Komponen
- ESP32-S3
- Motor Stepper 28BYJ-48
- Driver A4988

## Wiring → docs/images/wiring_motorstepper

## Program → src/motorstepper/motorstepper.ino

## Langkah Perrcobaan 
1. Buat New Project
2. Sesuaikan wiring 
3. Upload kode ke ESP32-S3 / jalankan di wokwi
4. Buka serial monitor
5. Tekan tombol run dan perhatikan output
6. Amati putaran pada arah tertentu
