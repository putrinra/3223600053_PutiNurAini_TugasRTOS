# Percobaan menggunakan Button

## Menguji input dari dua tombol menggunakan ESP32 S3

## Komponen
- 1x ESP32-S3
- 2x Push Button

## Wiring → docs/images/wiring_button

## Program → src/button/button.ino

## Langkah Perrcobaan 
1. Buat New Project
2. Sesuaikan wiring 
3. Upload kode ke ESP32-S3 / jalankan di wokwi
4. Buka serial monitor
5. Tekan tombol run dan perhatikan output
