# Percobaan Servo Motor

## Mengontrol posisi servo menggunakan sinyal PWM dari ESP32-S3.

## Komponen
- ESP32-S3
- Servo motor SG90

## Wiring → docs/images/wiring_servo

## Program → src/servo/servo.ino

## Langkah Perrcobaan 
1. Buat New Project
2. Sesuaikan wiring 
3. Upload kode ke ESP32-S3 / jalankan di wokwi
4. Servo akan bergerak dari sudut kecil ke besar secara periodik.
