# Percobaan Potensiometer dengan ESP32-S3

## Membaca nilai analog dari potensiometer menggunakan ADC ESP32.

## Komponen
- ESP32-S3
- Potensiometer

## Wiring → docs/images/wiring_potensio

## Program → src/potensio/potensio.ino

## Langkah Perrcobaan 
1. Buat New Project
2. Sesuaikan wiring 
3. Upload kode ke ESP32-S3 / jalankan di wokwi
4. Putar potensiometer.
5. Lihat nilai ADC di Serial Monitor.
