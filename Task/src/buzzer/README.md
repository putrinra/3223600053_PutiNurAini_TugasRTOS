# Percobaan Buzzer dengan ESP32-S3

## Menghasilkan bunyi menggunakan buzzer piezo melalui PWM ESP32.

## Komponen
- 1x ESP32-S3
- 1x Buzzer aktif

## Wiring → docs/images/wiring_buzzer

## Program → src/buzzer/buzzer.ino

## Langkah Perrcobaan 
1. Buat New Project
2. Sesuaikan wiring 
3. Upload kode ke ESP32-S3 / jalankan di wokwi
4. ESP32 mengeluarkan bunyi dari buzzer
