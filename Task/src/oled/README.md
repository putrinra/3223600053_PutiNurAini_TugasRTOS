# Percobaan OLED Display

## Menampilkan teks pada layar OLED menggunakan komunikasi I2C

## Komponen
- ESP32-S3
- OLED 0.96” (SSD1306)

## Wiring → docs/images/wiring_oled

## Program → src/oled/oled.ino

## Langkah Perrcobaan 
1. Buat New Project
2. Sesuaikan wiring 
3. Upload kode ke ESP32-S3 / jalankan di wokwi
4. Amati hasil di layar oled
